# React Learning Journey

A collection of React projects and exercises from the React Complete Guide course by Maximilian Schwarzmüller.

## Projects

- **08-refs-portals** - Learn about React refs and portals
- **09-project-management-project** - Build a project management application

## Getting Started

Each project has its own folder with its own `package.json`. To run a project:

```bash
cd <project-folder>
npm install
npm run dev
```

## Tech Stack

- React
- Vite
- JavaScript/JSX/TypeScript
